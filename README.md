# Dev Team Task Tracker

A premium, dark-mode task tracking application for development teams.

## Tech Stack
- **Core**: React + Vite
- **Styling**: TailwindCSS
- **Icons**: Lucide React
- **Motion**: Framer Motion

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev
```
