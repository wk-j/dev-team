# Settings Page Design

## Overview
The Settings page allows users to customize their experience, manage their profile, and configure application preferences.

*(Mockup generation pending)*

## Key Features

### 1. Sidebar Navigation
- **Profile**: Personal details and avatar.
- **Notifications**: Email and push notification preferences.
- **Theme**: Appearance settings.
- **Integrations**: Connections to other tools (GitHub, Slack).

### 2. User Profile
- **Avatar Upload**: Drag and drop area for profile picture.
- **Form Fields**: Name, Email, Role (read-only).

### 3. Theme Settings
- **Mode Toggle**: Light / Dark / System.
- **Accent Color**: Picker to choose the primary brand color (Blue, Purple, Green, etc.).
- **Reduced Motion**: Accessibility toggle.

## Interactions
- **Real-time Preview**: Theme changes apply immediately.
- **Save Feedback**: Toast notification on successful save.
