# Team Page Design

## Overview
The Team page provides a directory of all team members, their current status, and workload. It facilitates team management and collaboration.

## Visual Reference
![Team Page Mockup](assets/team_page_mockup.png)

## Key Features

### 1. Team Grid
- **Card Layout**: A grid of cards representing each team member.
- **Member Details**:
    - **Avatar**: Large profile picture.
    - **Name & Role**: Clear identification (e.g., "Jane Doe - Senior Dev").
    - **Status**: Online/Offline/Busy indicator.
    - **Workload**: Mini-chart or progress bar showing current task load.

### 2. Management Actions
- **Invite Member**: Button to add new users via email.
- **Search**: Find members by name or role.

### 3. Member Profile (Modal/Drawer)
- Clicking a card opens a detailed view showing:
    - Recent activity.
    - Assigned tasks.
    - Contact info.

## Interactions
- **Hover**: Cards scale up slightly.
- **Status Update**: Quick toggle for own status (if viewing self).
