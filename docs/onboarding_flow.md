## Accessibility Onboarding Path

### Screen Reader Mode
- All narration is spoken, not just displayed
- "Press Tab to navigate through options"
- Spatial descriptions: "Your star is now at the center"

### Reduced Motion Mode
- Skip particle animations
- Instant state transitions
- Same content, faster completion

### Classic View Users
- 2D diagram-based onboarding
- Same concepts, flat representation
- "This is your profile card. This is the team list."
